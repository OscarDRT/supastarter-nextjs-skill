# AI Patterns (Next.js)

AI features and models in supastarter Next.js.

## Patterns

- **SDK**: Vercel AI SDK; optional AI package in `packages/ai/` (e.g. `index.ts` for model config).
- **API**: Streaming or non-streaming procedures in `packages/api`; use AI SDK in procedure handlers.
- **Chatbot**: Example flows in docs; prompts and model config as needed.

## Key Paths

- AI package: `packages/ai/` (e.g. index.ts)
- API: `packages/api/` (streaming procedures)
